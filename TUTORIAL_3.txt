=================================================== models/hotel.py ===================================================
1.0. O que é database_insert(exec_, hotel_object)?
    -> Função que realiza uma verificação afim de saber se um objeto já existe no banco
    -> exec_        || É o cursor criado em "menu/id_1_query.py" e que torna consultas ao banco possível
    -> hotel_object || Parâmetro programado p/ receber um objeto de hotel, vindo do algoritmo "id_3_object_insertion.py"
                       Deste objeto, o atributo "hotel_id" é pêgo e verificado no função, através de um iteração
                       A função itera sob os dados do banco via atributo "hotel_id" e compara com o do objeto
                       Havendo similaridade de nomes, o objeto alvo é impedido de ser criado

============================================ menu/id_3_object_insertion.py ============================================
1.1. Algoritmo para criar um novo objeto do banco deste projeto
1.2. As funções principais estão especificadas no arquivo do algoritmo
1.3. A função principal 1 serve de escolha para criar um novo objeto ou encerrar o algoritmo
1.4. A função principal 2 serve para criar um novo objeto, informando instruções e fazendo tratamentos, se necessário
